Welcome to the Guessing Game! This game allows players to guess numbers generated by the computer, offering different difficulty levels, dynamic scoring, and a history of the best results.

## Features

### 1. Main Menu
The game features a main menu with the following options:
- **Start New Game**  
- **View Rules**  
- **View Score History**  
- **Exit** 🚪

### 2. Difficulty Levels
Players can choose from three difficulty levels:
- **Easy**: Guess a number between 1 and 50, with 10 attempts.
- **Medium**: Guess a number between 1 and 100, with 7 attempts.
- **Hard**: Guess a number between 1 and 200, with 5 attempts.

### 3. Scoring System
- **Base score by level**:
  - Easy: 100 points
  - Medium: 200 points
  - Hard: 300 points
- **Deductions**: Points are subtracted for each attempt.
- **Bonus**: +50 points for each unused attempt.

### 4. Score History
The game keeps track of the last 10 scores in an array, storing the score and the difficulty level chosen for each game.

### 5. Additional Features

#### 5.1 Sequence Mode 🔢
A special mode where the player must guess a sequence of 3 numbers instead of just one.

Example sequence: [5, 8, 2]  
The player must guess each number in the sequence correctly.

#### 5.2 Hints System 💡
Players can request hints for a cost in points:
- Parity Hint (Even/Odd): -10 points
- Range Hint (Upper/Lower Half): -20 points
- Proximity Hint (Hot/Cold): -15 points

#### 5.3 Record System 🏆
The game stores the best scores for each difficulty level.

## Technical Requirements

- **Arrays**: Used for storing score history and level configurations.
- **Loops**: A loop is used to keep the main menu active and to handle attempts.
- **Conditionals**: Used for validating user input and providing appropriate messages based on the player's guess.

# Jogo de Adivinhação 🎮

    <img src="https://cdn.universoracionalista.org/wp-content/uploads/2020/06/Alan-Turing-o-pai-da-ciencia-da-computacao-800x620.webp" alt="Alan Turing" width="50%">

Bem-vindo ao Jogo de Adivinhação! Este jogo permite que os jogadores adivinhem números gerados pelo computador, oferecendo diferentes níveis de dificuldade, pontuação dinâmica e um histórico das melhores pontuações.

## Funcionalidades

### 1. Menu Principal
O jogo possui um menu principal com as seguintes opções:
- **Iniciar Novo Jogo**  
- **Ver Regras**  
- **Ver Histórico de Pontuações**  
- **Sair** 🚪

### 2. Níveis de Dificuldade
O jogador pode escolher entre três níveis de dificuldade:
- **Fácil**: Adivinhar um número entre 1 e 50, com 10 tentativas.
- **Médio**: Adivinhar um número entre 1 e 100, com 7 tentativas.
- **Difícil**: Adivinhar um número entre 1 e 200, com 5 tentativas.

### 3. Sistema de Pontuação
- **Pontuação base por nível**:
  - Fácil: 100 pontos
  - Médio: 200 pontos
  - Difícil: 300 pontos
- **Descontos**: Pontos são subtraídos a cada tentativa.
- **Bônus**: +50 pontos para cada tentativa não utilizada.

### 4. Histórico de Pontuações
O jogo mantém um histórico das últimas 10 pontuações em um array, armazenando a pontuação e o nível de dificuldade escolhido para cada jogo.

### 5. Funcionalidades Adicionais

#### 5.1 Modo Sequencial 🔢
Um modo especial onde o jogador deve adivinhar uma sequência de 3 números em vez de apenas um.

Exemplo de sequência: [5, 8, 2]  
O jogador deve adivinhar corretamente cada número da sequência.

#### 5.2 Sistema de Dicas 💡
O jogador pode solicitar dicas com um custo em pontos:
- Dica de Paridade (Par/Ímpar): -10 pontos
- Dica de Intervalo (Metade superior/inferior): -20 pontos
- Dica de Proximidade (Quente/Frio): -15 pontos

#### 5.3 Sistema de Recordes 🏆
O jogo armazena as melhores pontuações para cada nível de dificuldade.

## Requisitos Técnicos

- **Arrays**: Usados para armazenar o histórico de pontuações e configurações dos níveis.
- **Loops**: Um loop mantém o menu principal ativo e gerencia as tentativas.
- **Estruturas de Seleção**: Usadas para validar entradas do usuário e fornecer mensagens adequadas com base no palpite.
